      <ul>
        <li><strong>Player Movement &amp; Mechanics:</strong> Implemented walking, running, slope movement, and double jump using Unity's character controller. Developed a third-person free-look and aim camera with smooth transitions via Cinemachine.</li>
        <li><strong>Elemental Ability System:</strong> Designed and implemented an elemental ability system — creating solid platforms, shooting fire, and controlling water — with save/load functionality to preserve progress.</li>
        <li><strong>User Interface:</strong> Created a generic cut scene manager for story delivery, and developed UI button animations and sound effects to enhance the interactive experience. Implemented a generic input system supporting both gamepad and keyboard/mouse controls.</li>
        <li><strong>Graphics &amp; Effects:</strong> Managed scene lighting and post-processing effects. Integrated game background music to enhance the atmosphere.</li>
        <li><strong>PlayStation Porting &amp; Optimisation:</strong> Ported to PS5, utilising haptic feedback and adaptive triggers for an immersive experience. Optimised assets and lighting to maintain high performance on console.</li>
        <li><strong>Project Management:</strong> Coordinated with a small team of game designers and artists. Used version control systems to manage project files and facilitate collaboration.</li>
